# Odin Recipes

A simple static website built using HTML as part of The Odin Project curriculum.

## Project Overview

This project contains a main index page with links to individual recipe pages.  
Each recipe page includes:

- Title
- Image
- Ingredients list
- Step-by-step directions

The goal of this project is to practice basic HTML structure and relative linking between pages.

## Skills Practiced

- HTML boilerplate structure
- Headings and paragraphs
- Unordered and ordered lists
- Images and alt attributes
- Relative file paths
- Folder structure organization
- Git and GitHub workflow
